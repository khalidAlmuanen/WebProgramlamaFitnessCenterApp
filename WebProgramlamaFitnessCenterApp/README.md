# WebProgramlamaFitnessCenterApp

Bu proje, Sakarya Üniversitesi Bilgisayar Mühendisliği bölümünde 
WEB PROGRAMLAMA dersi için geliştirilen bir Spor Salonu (Fitness Center) 
yönetim ve randevu sistemidir.

## Amaç
- Spor salonlarını, hizmetlerini ve antrenörlerini yönetmek
- Üyeler için randevu oluşturmak
- Yapay zekâ desteği ile egzersiz önerileri sunmak

## Kullanılan Teknolojiler
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5
- HTML5, CSS3, JavaScript
